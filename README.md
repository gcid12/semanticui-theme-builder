
## Running Example
Clone this repo:

```git clone git@github.com:gcid12/semanticui-theme-builder.git```

Install dependencies:

```yarn```

Move inside the project

```cd semanticui-theme-builder```


Start Project
```yarn start```

Open in your browser, port 3000

```http://localhost:3000/```
